
# Quickstart

[nodejs v22](https://nodejs.org/en) with [yarn](https://yarnpkg.com/) recommended.
```bash
yarn # install dependencies
yarn run dev
```

# Deploy

App published to https://retho.github.io/todoapp-pwa/ via [Github Pages](https://docs.github.com/en/pages)

```bash
yarn run deploy
```
